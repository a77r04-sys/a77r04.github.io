End Effector - Final Prototype

Optimized Gripping

The final prototype utilized a linear actuator and springs to create a gripping mechanism that could bend in more, allowing the end effector to better clamp onto rocks. Additionally, the end effector employed TPU grippers to adjust to rugged edges.

System Integration

The end effector had both a camera and a laser to make it easier for the controls team to use. This iteration successfully removed the heavy self-actuating screwdriver to save weight.
